How to upload a shapefile

  • Download and fill in the templates (.zip) or use your own shapefile

  • your ESRI shapefiles must be included in a single .zip file
  • the .zip file must contain the following files:
    • .shp (main file)
    • .shx (index file)
    • .dbf (database file)
  • each file must have the same prefix, for example:
    • locations.shp
    • locations.shx
    • locations.dbf
  • other file types can be included in the .zip file but are not essential
  • all shapefiles should use Ordnance Survey British National Grid references, for example, OSGB_1936_National_Grid
