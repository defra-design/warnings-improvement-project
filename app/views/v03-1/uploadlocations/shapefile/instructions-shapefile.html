How to upload a shapefile

You can upload your own shapefile or download the template shapefile and enter your information (.zip).

If you use the template the column names, or ‘fields’, will match the ones in the system automatically. If you use your own spreadsheet, you can match your fields to the ones in the system after you upload your file.

What format your shapefile should be in

  • your ESRI shapefiles must be included in a single .zip file
  • the .zip file must contain the following files:
    • .shp (main file)
    • .shx (index file)
    • .dbf (database file)
  • each file must have the same prefix, for example:
    • locations.shp
    • locations.shx
    • locations.dbf
  • other file types can be included in the .zip file but are not essential

What you need to include in your shapefile

Some fields must be included in your shapefile. Some fields are optional, to give more information about your locations and help you act when flooding happens.

  • location name
What information must I include?

How you refer to the location, for example: head office, Brayton Water pumping station
  • location reference
  • location type
  • location team
  • action to be taken
  • criticality
  • notes
What information should I include in the optional fields?

Your internal reference, for example: PS01, unit 57, Brid_04

For example, pumping station, ground floor flat, office, retail unit

The team with responsibility for that location, for example: national team, north-west team

What you can do to reduce the potential effects of flooding, for example: inspect the location, use sandbags, move stock, evacuate

How important the location is to your business, for example, low, medium, business critical

Any notes you may find helpful, for example, John Smith has the flood plan for this location, the last time this location flooded we used sandbags
